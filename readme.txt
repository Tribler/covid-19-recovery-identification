This is the SSI App that a user can use to receive and manage SSI passports.

# Running:
 1. Navigate to the "SSI-App" directory in your console of choice

 2. Run the command "yarn web" to run the app in webmode

 3. Step 2 should open two tabs in your browser:
	1. The Expo dev console, here you can look at the app's debug messages.
	2. The browser tab with your app running on it.

 4. To close the app stop execution on the console you used in Step 2.
