# ipv8-service

Integration of the IPv8 Service with Android, IOS and Web.

## Installation

```sh
npm install ipv8-service
```

## Usage

```js
import Ipv8Service from "ipv8-service";

// ...

const result = await Ipv8Service.multiply(3, 7);
```
